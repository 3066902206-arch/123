# iPhone Simulator Web App

这是一个模拟 iPhone 界面的 Web 应用，包含设置、应用图标自定义、API 配置等功能。

## 如何运行

1. 双击打开 `index.html` 文件即可在浏览器中运行。
2. 为了获得最佳体验，建议使用 Chrome、Edge 或 Safari 浏览器。

## 功能说明

### 1. 界面与交互
*   **模拟 iPhone 外观**：包含刘海、圆角屏幕、底部 Dock 栏。
*   **导航**：点击应用图标打开应用（目前仅“设置”有完整功能），点击底部的横条（Home Indicator）返回主屏幕。

### 2. 设置功能
点击桌面上的 **“设置”** 图标进入：

*   **个人信息**：
    *   点击顶部头像可上传更换头像。
    *   点击名称输入框可修改名称。
*   **API 配置**：
    *   输入 API 地址（例如 OpenAI 格式接口）和密钥。
    *   点击“配置”按钮拉取模型列表。
    *   选择模型后会自动保存。
    *   *(注：如果 API 请求因跨域 (CORS) 失败，系统会提示加载演示数据用于测试)*
*   **外观设置**：
    *   **壁纸**：点击预览框上传图片更换主屏幕壁纸。
    *   **应用图标**：列表显示所有应用，点击任意应用即可从相册选择图片替换该应用的图标。替换后实时生效。
*   **全屏模式**：
    *   开启开关后，手机外框消失，界面将铺满整个浏览器窗口。

### 3. 数据保存
所有设置（头像、壁纸、自定义图标、API配置等）均保存在浏览器的本地存储 (LocalStorage) 中，刷新页面后不会丢失。
